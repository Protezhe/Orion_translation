# Парковое радио — инструкция по установке

## Требования

- Windows 7 / 10 / 11
- Python 3.10 или новее — https://www.python.org/downloads/
  - При установке поставить галочку **"Add Python to PATH"**
- VLC media player — https://www.videolan.org/vlc/
  - Разрядность VLC должна совпадать с разрядностью Python (оба 64-bit или оба 32-bit)

---

## Установка

### 1. Установить Python

Скачать с https://www.python.org/downloads/ и запустить установщик.
Обязательно поставить галочку **"Add Python to PATH"** на первом экране.

### 2. Установить VLC

Скачать с https://www.videolan.org/vlc/ и установить.

### 3. Установить зависимости

Открыть папку с программой, в адресной строке проводника написать `cmd` и нажать Enter.
В открывшемся окне выполнить:

```
pip install -r requirements.txt
```

---

## Структура папок

```
park_radio_server.py   — основная программа
start.bat              — запуск
config.json            — настройки
requirements.txt       — зависимости
music\                 — сюда положить музыкальные файлы
announcements\         — сюда положить файлы объявлений
templates\             — файлы интерфейса (не трогать)
```

Поддерживаемые форматы: **MP3, WAV, OGG, FLAC, AAC, M4A, WMA**

---

## Запуск

Дважды кликнуть на **start.bat**

В открывшемся окне появится адрес для подключения:

```
На этом ПК:   http://localhost:8080
В локалке:    http://192.168.1.X:8080
```

Открыть любой браузер и перейти по адресу.
С телефона или другого компьютера в той же сети использовать адрес **В локалке**.

---

## Настройка

Все настройки доступны прямо в браузере:

| Настройка | Описание |
|---|---|
| Громкость музыки | Регулируется слайдером в реальном времени |
| Громкость объявлений | Регулируется слайдером в реальном времени |
| Частота объявлений | От 1 до 19 песен между объявлениями, или «никогда» |
| Расписание | Время автоматического включения и выключения |

Все настройки сохраняются автоматически в `config.json`.

## Возможные проблемы

**`python-vlc` не работает / ошибка при запуске**
Убедитесь что VLC установлен и его разрядность совпадает с Python.
Проверить разрядность Python: открыть cmd и написать `python --version`.
64-bit Windows → устанавливать 64-bit VLC и 64-bit Python.

**Порт 8080 занят**
В файле `park_radio_server.py` в последней строке `app.run(...)` изменить `port=8080` на другой, например `port=8888`.

**Нет звука**
Проверить что файлы лежат в папке `music\` и имеют поддерживаемый формат.
Проверить громкость в системе и в интерфейсе радио.
